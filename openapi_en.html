Public Information

Exchange Interface Document

Document Version: v1.2.2_23-05-30
Interface Protocol: http/https
Message format: json
Message Encoding: utf-8
Spot Interface Address: https://oapi.websea.com/
Futures Interface Address: https://coapi.websea.com/

Authentication Authorization

About token, secret-key application and modification, please carry out relevant operations on the "Personal Center - Open API" page.

Interface Permission Validation

The client requests the transaction interface, and the header must include the Nonce, token, Signature parameters. For example:

$header[] = 'Nonce: 1534927978_ab43c';
$header[] = 'Token: 57ba172a6be125cca2f449826f9980ca';
$header[] = 'Signature: v490hupi0s0bckcp6ivb69p921';
curl\_setopt\($ch\, CURLOPT\_HTTPHEADER\, $header\);

token：

User token, please check in "Personal Center - Open Interface".

Nonce：

Random number, the client generates random characters each time it requests. Generation rule, timestamp_5 random letters or numbers. Such as: 1534927978_ab43c
Note: The time difference cannot exceed 60 seconds, and the same random number can only be used once.

Signature：

Signature, calculated using $token,$secret_key,$nonce,$_POST,$_GET, server verifies the signature to determine if the request is from the user.

Python example code for generating Signature:

def sign(Nonce,data=dict()):

tmp = list()
tmp.append(TOKEN)
tmp.append(SECRET)
tmp.append(Nonce)
for d,x in data.items():
	tmp.append(str(d) + "=" + str(x))

return hashlib.sha1(''.join(sorted(tmp))).hexdigest()

Generating Signature's PHP sample code:

private function signature($nonce)
{
	$token = TOKEN;
	$secret_key = SECRET-KEY;
	$tmpArr = array($api_key,$secret_key,$nonce);
	foreach($_GET as $k=>$v){
		$tmpArr[] = $k."=".$v;
	}
	foreach($_POST as $k=>$v){
		$tmpArr[] = $k."=".$v;
	}
	sort($tmpArr, SORT_STRING);
	$tmpStr = implode( $tmpArr );
	$signature = sha1( $tmpStr );
	return $signature;
}
