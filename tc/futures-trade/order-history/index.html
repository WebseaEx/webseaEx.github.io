    查詢歷史委託列表

    地址: openApi/contract/historyList

    設計審核

    未審核

    接收參數

    需要

    提交方式:get

    範例

    提交參數

    參數名 是否必需 類型 示例 說明
    symbol 否 string 交易對(當前交易對必傳,全部交易對不傳)
    from 否 int 122 查詢起始order_id
    direct 否 string prev 查詢方向(預設 prev)，prev 向前，時間（或 ID）倒序；next 向後，時間（或 ID）正序）。（舉例一列數：1，2，3，4，5。from=4，prev有3，2，1；next只有5）
    limit 否 int 20 分頁返回的結果集數量，默認為20，最大為100(具體參見分頁處的描述)
    is_full 否 int 1 1逐倉 2全倉

    返回參數

    參數名 是否必需 類型 範例 說明
    errno 是 int 0 返回狀態碼
    errmsg 是 string 成功 返回資訊
    result 是 object 見示例 返回數據

    result參數

    參數名 是否必需 類型 示例 說明
    order_id 是 string 11535 訂單號
    ctime 是 int 1576746253 委託時間
    symbol 是 string EOS-USDT 交易對
    price 是 string 14 委託價格
    price_avg 是 string 15 成交均價
    lever_rate 是 int 10 桿杆倍數
    amount 是 string 150 委託數量
    deal_amount 是 string 100 已成交
    type 是 string buy-limit 委託類型
    status 是 int 3 訂單狀態（1=掛單中,2=部分成交,3=已成交,4=撤銷中,5=部分撤銷,6=已撤銷）
    contract_type 是 string open 開倉 (open開倉 close平倉)
    profit 是 string 10 總盈虧

    返回範例

    JSON
    {
    "errno": 0,
    "errmsg": "success",
    "result": [
        {
            "order_id": "11535", //訂單號
            "ctime": 1576746253, //委託時間
            "symbol": "EOS-USDT", //交易對
            "price": "14", //委託價格
            "price_avg": "15", //成交均價
            "lever_rate": 10, //槓桿倍數
