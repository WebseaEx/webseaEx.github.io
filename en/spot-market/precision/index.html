    Query Trading Pair Precision

    Address/openApi/market/precision

    Receiving Parameters

    Required

    Submission method: GET

    Example

    curl -d /openApi/market/precision

    Submission Parameters

    Text Only
    | Parameter Name | Required | Type | Example | Description |
    | -- | -- | -- | -- | -- | | symbol | 否 | string | BTC-USDT | Trading pair, optional, if not filled, the precision of all trading pairs is returned |

    Return Parameters

    Parameter Required Type Example Description
    errno 是 int 0 Status code
    errmsg 是 success success Status Information
    result Yes object See example List

    result Parameters Explanation

    Parameter Name Required Type Example Description
    amount is string LINK-BTC Quantity Precision
    price is string LINK-BTC price precision
    minQuantity is string 0.01 Minimum order quantity
    maxQuantity 是 string 100000000 Maximum order quantity
    minPrice is string 0.01 Minimum order price
    maxPrice 是 string 100000000 Maximum order price

    Return Example

    JSON
    {
    "errno":0,
    "errmsg":"success",
    "result":{
        "BTC-USDT":{
            "amount":"3",
            "minQuantity":"0.01",
            "maxQuantity":"100000000",
            "price":"2",
            "minPrice":"0.000001",
            "maxPrice":"100000"
        }
    }
}
