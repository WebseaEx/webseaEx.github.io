インターフェイス権限の検証

クライアントはトランザクションインターフェースを要求し、headerにはすべてNonce,token,Signatureパラメータを含める必要があります。例えば：

$header[] = 'Nonce: 1534927978_ab43c';
$header[] = 'Token: 57ba172a6be125cca2f449826f9980ca';
$header[] = 'Signature: v490hupi0s0bckcp6ivb69p921';
curl\_setopt\($ch\, CURLOPT\_HTTPHEADER\, $header\);

token：

ユーザートークン、"個人センター - オープンAPI"をご覧ください。

Nonce：

ランダム数値、クライアントは毎回リクエストでランダムに文字を生成します。生成規則、タイムスタンプ_5文字のランダムなアルファベットまたは数字。例：1534927978_ab43c
注意：時間誤差は60秒を超えることはできません、同じランダム数は一度しか使用できません。

Signature：

署名、$token、$secret_key、$nonce、$_POST、$_GETを使用して計算され、サーバーは署名を確認してリクエストがユーザーから来たものかどうかを確認します。

Signatureを生成するPythonの例示コード：

def sign(Nonce,data=dict()):

tmp = list()
tmp.append(TOKEN)
tmp.append(SECRET)
tmp.append(Nonce)
for d,x in data.items():
	tmp.append(str(d) + "=" + str(x))

return hashlib.sha1(''.join(sorted(tmp))).hexdigest()

Signatureを生成するPHPの例示コード：

private function signature($nonce)
{
	$token = TOKEN;
	$secret_key = SECRET-KEY;
	$tmpArr = array($api_key,$secret_key,$nonce);
	foreach($_GET as $k=>$v){
		$tmpArr[] = $k."=".$v;
	}
	foreach($_POST as $k=>$v){
		$tmpArr[] = $k."=".$v;
	}
	sort($tmpArr, SORT_STRING);
	$tmpStr = implode( $tmpArr );
	$signature = sha1( $tmpStr );
	return $signature;
}
